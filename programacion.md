# Parte Teórica - Programación en Java

## Tipos de datos primitivos

Java tiene 8 tipos primitivos:

- byte
- short
- int
- long
- float
- double
- char
- boolean

---

## Estructuras de control

Permiten controlar el flujo del programa.

- if / else → decisiones.
- switch → múltiples opciones.
- for → repetición con contador.
- while → repetición con condición.
- do-while → ejecuta al menos una vez.

---

## Programación Orientada a Objetos (POO)

Paradigma basado en clases y objetos.

### Cuatro pilares:

1. Encapsulamiento
2. Abstracción
3. Herencia
4. Polimorfismo

---

## Herencia en Java

Se utiliza la palabra clave:

extends

Permite que una clase herede atributos y métodos de otra.

---

## Modificadores de acceso

- public
- private
- protected
- default

---

## Variables de entorno

Son configuraciones del sistema operativo que permiten ejecutar programas como Java desde la terminal.
